# @padloc/cordova

This package contains the sources to build the mobile apps for Padloc.

## Requirements

-   Java (1.8+) or OpenJDK (11+)
-   Android Studio, with:
    -   Android API 30
    -   Android SDK Build tools 30 + 31 (32 was default)
    -   Android SDK CLI Tools
-   Gradle (7.2, specifically)
